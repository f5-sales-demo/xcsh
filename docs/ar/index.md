xcsh هي واجهة سطر أوامر للتطوير مدعومة بالذكاء الاصطناعي تضم وكيل ترميز TypeScript وطبقة Rust الأصلية (`pi-natives`). تمتد على خط المصدر المفتوح
[`badlogic/pi-mono`](https://github.com/badlogic/pi-mono) بإضافة بيئة تشغيل مُحصَّنة، وجلسات طويلة الأمد مع التنقل الشجري والضغط، وأداة Python IPython، ودعم MCP الكامل، ونظام مهارات، وتغليف المنصة يستهدف Linux وmacOS وWindows.

## من أين تبدأ

- **[سياقات F5 XC](/runtime-tools/context-command)** — الاتصال بمستأجري F5 Distributed Cloud.
  إنشاء السياقات، والتبديل بينها، وإدارة مساحات الأسماء والبيانات الاعتمادية.
- **التهيئة** — كيفية اكتشاف xcsh للتهيئة وتحليلها وتطبيقها بالتدرج.
- **بيئة التشغيل والأدوات** — بيئات تشغيل أدوات bash / notebook / resolve وواجهة الأوامر المسبوقة بـ slash.
- **الجلسات** — سجل الإدخال للإلحاق فقط، والتنقل الشجري، والضغط، ونظام الذاكرة المستقل.
- **الطبقة الأصلية (Rust)** — معمارية الإضافة N-API الخاصة بـ `pi-natives` التي تشغّل shell / PTY / الوسائط / البحث.
- **MCP** — التهيئة، والبروتوكول الداخلي، ودورة حياة بيئة التشغيل، وكيفية تأليف الخوادم والأدوات.
- **الامتدادات والمهارات والإضافات** — التأليف، والتحميل، وقواعد المطابقة، والسوق، ومثبِّت الإضافات.
- **الموفرون والنماذج** — تهيئة النماذج، والدفق الداخلي، وبيئة تشغيل Python / IPython.
- **واجهة المستخدم النصية (TUI)** — السمات، وأمر `/tree`، وخطافات التكامل للامتدادات والأدوات المخصصة.
